# Wix Registration System

A module for managing Wix-based registration flows and group invitations for the SeeC platform.

## Overview

This package provides functionality to:
- Handle user registration workflows
- Integrate with Wix Groups API
- Manage group invitations and assignments
- Process user profile data for group matching

## Installation

```bash
npm install @seec/wix-registration-system
```
